# Network Packet Sniffer

A Python-based Network Packet Sniffer that captures and analyzes network traffic in real time. This project helps users understand how data travels across networks by inspecting packets and displaying protocol-level information.

## Features

- Capture network packets in real time
- Display source and destination IP addresses
- Analyze packet headers and protocols
- Monitor network traffic activity
- Lightweight and easy to understand
- Educational tool for networking and cybersecurity

## Technologies Used

- Python 3
- Socket Programming
- Raw Sockets
